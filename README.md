# None-cli

## 功能介绍

1. 自动快速生成项目模板，目前只支持Vue3的项目模板

2. 快速构建项目

3. 集成默认webpack优化配置打包项目

4. 支持开启Vue2.x按需热更新本地开发环境

## 如何使用

运行以下命令： 

```

    npm install none-cli

```

### 查看cli当前版本信息

```

    none -V

```

### 快速生成项目模板

```

    none init my-project-name

```

### 开启Vue3本地服务（自动开启热更新）

```

    none dev

```

### 开启Vue2.x本地服务

```

    none dev --v2

```

### 打包项目文件

```

    none build

```

### 运行测试用例(尚未完善)

```

    none test

```

### 快速进行git commit

```

    none commit

```
