# None-cli

## 功能介绍

1. 自动快速生成项目模板，目前只支持vue版本的项目模板

2. 快速构建项目

3. 一键运行测试用例

## 如何使用

运行以下命令： 

```

    npm install none-cli

```

### 查看cli当前版本信息

```

    none -V

```

### 快速生成项目模板

```

    none init my-project-name

```

### 开启本地服务（自动开启热更新）

```

    none dev

```

### 打包项目文件

```

    none build

```

### 运行测试用例

```

    none test

```

### 快速进行git commit

```

    none commit

```
